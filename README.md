# Saw Rent

Saw Rent is a chainsaw rental web app built with React and Vite.

## Overview

This app is designed to support a real rental workflow with:

- Public renter-facing booking/request flow
- Admin-only management area
- Inventory tracking
- Booking management
- Customer tracking
- Maintenance status tracking
- Local persistence for app data

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
